# ShipOS SEO + GEO Execution Checklist

Use this as a working checklist. Mark each task as done with `[x]`.

## 1) Foundation (Week 1)

- [x] Define 1 primary ICP (for example: founders, creators, agencies) and 1 secondary ICP.
- [x] Define 3 core keyword clusters:
  - [x] AI social media scheduler
  - [x] LinkedIn scheduler / generator
  - [x] Social media content calendar
- [x] Create a page-to-keyword map (1 primary keyword per page).
- [x] Record baseline metrics in one sheet (clicks, impressions, avg position, organic signups).
- [x] Confirm tracking is active:
  - [x] Google Search Console
  - [x] GA4
  - [x] Bing Webmaster Tools

## 2) Technical SEO Hygiene (Week 1-2)

- [x] Verify key pages are indexable (no accidental `noindex`).
- [x] Ensure canonical tags are correct on landing pages.
- [x] Validate `sitemap.xml` includes all SEO pages and excludes junk URLs.
- [x] Verify `robots.txt` does not block important paths.
- [x] Ensure unique `<title>`, H1, and meta description for each commercial page.
- [x] Check Core Web Vitals (mobile-first) and fix highest-impact issues.
- [x] Confirm prerender output is visible to bots for JS-heavy pages.

## 3) GEO Readiness (AI Recommendation) (Week 2)

- [x] Add JSON-LD schema to core templates:
  - [x] `Organization`
  - [x] `SoftwareApplication` or `Product`
  - [x] `FAQPage` (where relevant)
  - [x] `BreadcrumbList`
- [x] Publish a clear "What is ShipOS?" explainer page with concise facts.
- [x] Publish a transparent pricing and plan explainer page.
- [x] Add strong FAQ sections with direct factual Q/A on key pages.
- [x] Publish a "ShipOS vs alternatives" comparison methodology page (objective criteria).

## 4) Commercial SEO Pages (Week 2-5)

- [x] Build or upgrade core money pages:
  - [x] `/ai-social-media-scheduler`
  - [x] `/linkedin-scheduler`
  - [x] `/social-media-calendar-tool`
- [x] Create 3 to 5 comparison pages:
  - [x] `/shipos-vs-buffer`
  - [x] `/shipos-vs-hootsuite`
  - [x] `/shipos-vs-later` (or relevant competitor)
- [x] Create 3 to 5 use-case pages:
  - [x] For SaaS founders
  - [x] For agencies
  - [x] For personal brands
- [x] Ensure every page has:
  - [x] Clear CTA
  - [x] Screenshots / Demo video
  - [x] Outcome-driven copy
  - [x] Internal links to tools, pricing, and signup

## 5) Free Tools Growth Loop (Week 3-6)

- [x] Prioritize top free tool pages by search potential (LinkedIn Previewer, Limit Checker).
- [x] For each prioritized tool page, add:
  - [x] "How to use" section
  - [x] Examples or templates
  - [x] FAQ
  - [x] Internal links to paid features
  - [x] Comparison snippet ("free tool vs full scheduler")
- [x] Add a "related tools" block to each tool page for internal-link depth.
- [x] Ensure each tool page targets one intent keyword and one variant.

## 6) Authority and Off-Page (Week 4-8)

- [ ] Set up and optimize external entity profiles with consistent brand description:
  - [ ] Product Hunt
  - [ ] G2 or Capterra (if applicable)
  - [ ] LinkedIn company page
- [ ] Publish 2 to 4 case studies with concrete outcomes.
- [ ] Publish 1 data-backed asset (benchmark, report, or templates) for citations.
- [ ] Run link outreach for high-quality placements (guest posts, founder interviews).
- [ ] Track referring domains and links to money pages (not just homepage links).

## 7) On-Page Conversion (Continuous)

- [x] Add above-the-fold value proposition and CTA on all SEO landing pages.
- [x] Add trust proof (testimonials, logos, outcomes, usage stats).
- [x] Match CTA by page type (tool, comparison, use-case).
- [x] Add clear next step path (free tool -> signup -> onboarding).

## 8) Measurement and Iteration (Weekly)

- [ ] Review Search Console query gains and losses every week.
- [ ] Refresh pages with rising impressions but low CTR (rewrite title and meta).
- [ ] Improve pages ranking in positions 6 to 20 first (quick wins).
- [ ] Prune or merge thin pages that do not gain traction.
- [ ] Expand only keyword clusters that prove conversions.

## Weekly KPI Checklist

- [ ] Non-brand organic clicks
- [ ] Impressions and avg position by cluster
- [ ] Indexed pages vs submitted pages
- [ ] Organic signup volume
- [ ] Landing-page conversion rate
- [ ] AI mention and citation checks (manual prompt tests)

## Day 90 Definition of Done

- [ ] 10 to 20 high-intent pages live and indexed
- [ ] Top free tools upgraded with strong conversion paths
- [ ] Schema deployed across key page templates
- [ ] 2 to 4 case studies + 1 citation-worthy data asset live
- [ ] Upward trend in non-brand clicks and organic signups for 6+ weeks
