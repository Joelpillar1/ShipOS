# ShipOS

ShipOS is an AI-Powered Social Media Command Center.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
